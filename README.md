Convertor MS0 to TXT
===========================================

Конвертор ПФР базы (MS0 версия 4 и 000 версия 3) в TXT читабельный формат файла, для дальнейшего импорта в Excel

Установка:
----------
Компиляция C# (.cs) файла возможна на любом ПК с предустановленной .NET Framework 2.0 или выше
```
SET FWPATH=%windir%\Microsoft.Net\Framework\v2.0.50727
%FWPATH%\csc.exe /out:Convertor_MS0_to_TXT.exe /t:exe "Convertor MS0 to TXT\*.cs"
```

Инструкция по запуску:
----------------------

- написано в самой программе (консольная программа)
